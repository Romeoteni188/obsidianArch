```bash
tar -xvzf lazygit_0.44.1_Linux_x86_64.tar.gz
```

Mover al directorio bin para que a nivel general se pueda usar la herramienta lazygit

```bash
sudo cp lazygit /usr/bin 
```

Con con el comando lazygit para usar la herramienta donde esta el codigo fuente.

```bash
lazygit
```
