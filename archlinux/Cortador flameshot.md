## flatpak

como instalarlo

```bash
sudo pacman -S flatpak
```

para mover los archivos 

```bash
sudo mv ~/Documents/icons/nombre_de_carpeta /usr/share/icons/
```

mover todo lo que esta dentro la carpeta

```bash
sudo mv ~/Documents/icons/* /usr/share/icons/
```
