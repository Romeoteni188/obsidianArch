como instalar spotify

comando con paru 

```bash
paru -S spotify
```

o tambien con yay 

```bash
yay -S spotify
```

te muestra un listado del repositorio al cual instalar 

 1) spotify  2) spotify-1.1  3) spotify-dev  4) spotify-edge  5) spotify-snapstore
 se escogio la opcion 2 y aceptar todo lo con y para la instalacion de spotify.
